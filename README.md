# Solar Panel Detection Pipeline

## Overview
This repository contains a complete pipeline for detecting rooftop solar panels
from satellite imagery using YOLOv8 instance segmentation.

## Folder Structure
- Pipeline code/: Inference pipeline
- Environment details/: Environment setup
- Trained model file/: YOLOv8 model weights
- Model card/: Model documentation
- Prediction files/: Output predictions
- Artefacts/: Sample images
- Model Training Logs/: Training metrics
